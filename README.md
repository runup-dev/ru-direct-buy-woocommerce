## 소개
우커머스 구매 프로세스는 장바구니 담기 > 장바구니 확인 > 체크아웃의 순으로 진행됩니다   
사용자 인터페이스를 고려하여 단일 제품을 즉시 체크아웃할수 있는 플러그인이 필요했습니다    
워드프레스 플러그인 제작방법에 대한 가이드라인정도를 제공했다 정도로 생각해주세요    

다만 위 기능을 구현한 국내 플러그인이 많이 없기는 하더군요 -_- ;
